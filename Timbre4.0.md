# Introducción

Esta es la documentación para la API de facturación electrónica para la versión 4.0 del SAT mediente el servicio de Kubox.mx.

Nuestra API tiene un diseño basado en REST, es orientada a recursos y usa respuestas HTTP, todas las respuestas de la API están en formato JSON, incluyendo errores.
